Directly promote Systeme.io affiliate offers with a professional floating badge and drop-up menu.

== Description ==

This plugin allows Systeme.io affiliates to display a professional promotional widget in the corner of their website. It features a floating badge that opens a drop-up menu containing hardcoded affiliate links.

Features:
* Easy configuration: Only enter your Affiliate ID.
* Language support: Choose between French and English.
* Customizable: Select built-in badges or upload your own.
* Responsive: Works on desktop (hover) and mobile (tap).
* Performance: Lightweight vanilla JavaScript and CSS.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Systeme.io Badge' in the dashboard to configure your settings.
